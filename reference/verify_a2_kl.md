# Verify A2-KL Optimization

Runs verification tests on the A2-KL optimization algorithm.

## Usage

``` r
verify_a2_kl(verbose = TRUE)
```

## Arguments

- verbose:

  Logical; if TRUE, print detailed results.

## Value

Logical; TRUE if all tests pass.

## Examples

``` r
if (FALSE) { # \dontrun{
verify_a2_kl()

} # }
```
