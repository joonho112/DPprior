# Publication-Quality Theme for DPprior Plots

Publication-Quality Theme for DPprior Plots

## Usage

``` r
theme_DPprior(base_size = 11, base_family = "")
```

## Arguments

- base_size:

  Numeric; base font size (default: 11).

- base_family:

  Character; base font family (default: "").

## Value

A ggplot2 theme object.
