# Normalize and Validate a Target PMF on {1, ..., J}

Normalize and Validate a Target PMF on {1, ..., J}

## Usage

``` r
.a2_kl_normalize_pmf(target_pmf, J)
```

## Arguments

- target_pmf:

  Numeric vector; unnormalized target PMF.

- J:

  Integer; sample size.

## Value

Numeric vector; normalized PMF of length J (support k=1,...,J).
