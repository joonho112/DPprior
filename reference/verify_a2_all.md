# Run All A2-MN Verification Tests

Comprehensive verification suite for the A2-MN Newton solver.

## Usage

``` r
verify_a2_all(verbose = TRUE)
```

## Arguments

- verbose:

  Logical; if TRUE, print detailed results.

## Value

Logical; TRUE if all tests pass.

## Examples

``` r
if (FALSE) { # \dontrun{
verify_a2_all()

} # }
```
